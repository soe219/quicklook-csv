Two Faces of QLQuickCSV

A Quick Look extension and a standalone app, sharing one parsing engine.

☵

Extension Quick Look

Press Space on any CSV in Finder and get an instant interactive data table.

  • Interactive data table with sticky headers
  • Type badges on every column header
  • Click-to-sort on any column
  • Statistics panel with min/max/mean
  • Zero launch time — native QLPreviewReply
⚗

App Host App

A full file navigator for power users who live in CSV data all day.

  • Sidebar file navigator with recent files
  • Session persistence across launches
  • Tabbed windows for comparing datasets
  • Drag-and-drop file opening
  • Settings sync with the extension via CFPreferences

The Pipeline

Five stages from raw bytes to a rendered data table.

Step 1

Read File

Detect encoding from UTF-8, UTF-16LE, UTF-16BE, and ASCII. Handle BOM markers.

→
Step 2

Detect Delimiter

Score candidates (comma, tab, semicolon, pipe) by column-count consistency.

→
Step 3

Parse CSV

RFC 4180 state machine handles quoted fields, escaped quotes, and newlines in values.

→
Step 4

Analyze Types

Sample rows to classify each column as INT, DECIMAL, DATE, BOOL, TEXT, or MIXED.

→
Step 5

Generate HTML

Emit a self-contained HTML document with inline CSS, sorting JS, and statistics.

Type Detection

Every column is classified automatically so you can spot data quality at a glance.

INT

Integer

42, -7, 1000000
DECIMAL

Decimal

3.14, -0.5, 99.99
DATE

Date

2026-03-01, 01/15/24
BOOL

Boolean

true, false, yes, no
TEXT

Text

"hello world"
MIXED

Mixed

42, hello, true
A MIXED badge signals a data quality issue — the column contains values of incompatible types, which often indicates malformed data, missing values, or inconsistent entry.

Features

Ten capabilities packed into one lightweight extension.

▦

Sticky Headers

Column headers stay pinned at the top as you scroll through thousands of rows.

✎

Auto-Detect Delimiter

Scores comma, tab, semicolon, and pipe to pick the best column separator.

★

Type Badges

Color-coded badges on every column header reveal the detected data type.

⇵

Column Sorting

Click any header to sort ascending or descending. Type-aware comparisons.

Σ

Statistics Panel

Toggle a panel showing min, max, mean, median, and null count per column.

⚙

Filter by Value

Click any cell value to instantly filter the table to matching rows.

▣

4 View Modes

Table, compact, card, and raw views to suit different inspection needs.

⌨

Multi-Encoding

Handles UTF-8, UTF-16LE, UTF-16BE, and ASCII with BOM detection.

☰

Header Auto-Detection

Heuristically determines whether the first row is a header or data.

⚡

Large File Handling

Streams and samples large files so previews stay fast even at 100K+ rows.
